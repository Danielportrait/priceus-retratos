# priceus (retratos, casamentos)

Price Us

Price Us é uma plataforma inovadora criada para ajudar fotógrafos a gerenciar e automatizar a criação de orçamentos personalizados para seus clientes. Este projeto foi idealizado e está sendo desenvolvido por mim, [Seu Nome], com o objetivo de oferecer uma solução prática e eficiente para profissionais do mercado de fotografia.

Sobre o Projeto

O Price Us foi concebido para facilitar o processo de interação entre fotógrafos e seus clientes, fornecendo ferramentas como:

Criação de orçamentos automatizados e personalizados.
Integração com Google Calendar, Google Maps e Google Flights para melhor planejamento de eventos e deslocamentos.
Gestão de assinaturas via gateway de pagamento integrado.
Painéis dedicados para fotógrafos e administradores, oferecendo controle total sobre eventos, orçamentos e configurações.
Tecnologias Utilizadas

Frontend: React.js
Backend: Node.js
Banco de Dados: MySQL
Hospedagem: AWS
Estrutura do Projeto

Painel do Fotógrafo: Ferramentas para personalizar orçamentos, gerenciar eventos e acessar configurações.
Área Administrativa: Controle completo sobre os dados da plataforma, assinaturas e relatórios.
